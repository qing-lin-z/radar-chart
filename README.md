# 雷达图生成器 / Radar Chart Generator

一个功能丰富的纯前端雷达图（Radar Chart）生成工具，支持图片智能识别、多维数据编辑和多种格式导出。

## ✨ 功能特性

- 📷 **智能识别导入** — 上传雷达图截图，自动识别维度名称和数值，支持手动校正
- 📊 **多数据系列** — 支持添加多个数据系列进行对比，每个系列可独立配置颜色
- 🎨 **15 种配色** — 内置 Apple 风格配色方案
- ⚙️ **灵活编辑** — 维度数量 3-12 可调，每个维度名称和数值可单独编辑
- 🖼️ **导出功能** — 支持导出 PNG、SVG、JSON 三种格式
- 🎯 **交互悬浮** — 鼠标悬停节点查看详细数值

## 🚀 使用方法

直接在浏览器中打开 index.html 即可使用，无需任何依赖。
